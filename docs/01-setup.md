# Setup local (Linux)

## Requisitos

- Node.js (ideal: LTS)
- npm

## Correr la app

Desde la raíz del repo:

```bash
cd web
npm install
npm run dev
```

Luego abrí `http://localhost:3000`.

## Estructura rápida

- `web/`: frontend PWA (Next.js App Router)
- `docs/`: guías de aprendizaje del proyecto

## Hecho

- Podés ver la app andando en el navegador.
