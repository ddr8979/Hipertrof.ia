# Índice (aprender programando hipertrof.ia)

Este repo está pensado para que aprendas a programar “en la era moderna”:
con **TypeScript**, **Next.js**, **Tailwind** y usando **IA** como copiloto, pero con criterio.

## Cómo leer esta documentación

- Leé en orden.
- Cada documento termina con “Hecho” (algo concreto en el repo).
- Si te trabás: copiá el error exacto y pedile a la IA *un plan*, no “la solución mágica”.

## Ruta de aprendizaje (MVP Mes 1)

1. `docs/01-setup.md` — levantar el proyecto local
2. `docs/02-arquitectura.md` — cómo está armado y por qué
3. `docs/03-flujo-de-trabajo-con-ia.md` — prompts útiles, cómo pedir cambios, cómo revisar
4. `docs/04-ui-identidad.md` — colores/fuentes del manual + estructura UI
5. `docs/05-feature-calorias.md` — Harris–Benedict (primer feature real)
6. `docs/06-base-de-datos-para-ninios.md` — DB y Prisma sin vueltas
7. `docs/07-mapa-completo-web.md` — dónde está cada módulo construido

## Reglas de oro (para aprender bien)

- **No copies/pegues sin entender**: pedí que te explique *línea por línea* lo que cambia.
- **Una feature por vez**: commit mental (aunque no uses git aún).
- **Primero la forma, después el detalle**: tipos, modelos, pantallas, luego validaciones y edge cases.
