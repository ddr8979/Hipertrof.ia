# Feature 1: calorías (Harris–Benedict)

## Objetivo

Implementar una pantalla que, con datos básicos del perfil, calcule:

- **BMR** (metabolismo basal)
- **TDEE** (gasto diario estimado con factor de actividad)

Esto es el “punto de partida” del manual.

## Qué vas a aprender haciendo esto

- Formularios en React
- Validación simple
- Funciones puras (testables)
- Tipos en TypeScript

## Hecho (cuando lo implementemos)

- Ruta `web/src/app/calorias/page.tsx`
- Función pura `web/src/lib/harrisBenedict.ts`
